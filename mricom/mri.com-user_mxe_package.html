MXE幹と各パッケージ

MXE幹はテンプレートであり、各実験における実際の作業はそれを基に作成したローカルなリポジトリ(個別の実験パッケージ)で行います。 MXE幹が骨格、ローカル・リポジトリで肉付け、という関係になります。

MXE幹: 全ての実験の共通部分に当たるもので以下を含む。

  • 実験環境の基本的なディレクトリ構造
  • 特定の目的で使えるFortranプログラム
  • 上を動かすスクリプトのサンプル
    • ディレクトリ「sample」の下にあるので、コピーして使ってください。

ローカル・リポジトリ: MXE幹をベースにして、さらに以下を含む。

  • 現在行っている実験の設定
  • 現在行っている実験にだけ必要と思われるツール類
  • 前処理、後処理、解析が再現できる、すぐに動かせるスクリプト

新しい実験パッケージはMXE幹から作り始めるのに対し、既存の実験を引き継ぐ場合には個別実験パッケージをコピー(かgit clone)します。
