日本沿岸海洋再解析データセット MOVE/MRI.COM-JPN Dataset

気象庁気象研究所によって開発された海洋モデル・データ同化システムを用いて、日本沿岸海洋再解析データセット「MOVE/MRI.COM-JPN Dataset」を作成しました。 このデータセットは、日本沿岸全域の海洋環境を水平解像度約2kmで再現しています。 また、作成に用いたシステムは、2020年10月28日から気象庁において現業運用されます。

概要

  • データセット名：MOVE/MRI.COM-JPN Dataset
  • 作成者：気象庁気象研究所 全球大気海洋研究部
  • 連絡先：気象研究所 全球大気海洋研究部 広瀬成章 jpninfo@mri-jma.go.jp
  • 手法：海洋モデル・データ同化システム「日本沿岸海況監視予測システム」による再解析値
  • 現在のバージョン：ver.2.0
  • 引用先: 広瀬成章, 坂本圭, 碓氷典久, 山中吾郎, 高野洋雄, 日本沿岸海況監視予測システム10年再解析値 (JPN Atlas 2020), 気象研究所技術報告, 83, 2020. DOI: 10.11483/mritechrepo.83

データ内容

  • 解析領域：117E-160E, 20N-52N
  • 解像度：水平1/33°×1/50° (約2km)
  • 解析期間：2008/01/01 - 2019/12/31 (New 2018-2019の延長)
    • 4本の実験(2008-2012,2013-2015,2016,2017)をまとめたもの
    • 2013/01/01, 2016/01/01, 2017/01/01 に不連続あり
  • 物理要素
    • 1日毎(平均)
      • 海洋3次元：流速uv、水温、塩分、鉛直拡散係数
      • 海洋2次元: 海面高度、鉛直積分流量、海面地衡流、海面地衡流2乗、海面相対渦度
      • 海面フラックス: 短波放射、長波放射、潜熱フラックス、顕熱フラックス、淡水フラックス(E,P,R)、塩分フラックス、風応力
      • その他: 海峡通過流量(対馬、津軽、間宮、宗谷)
      • 海氷: 密接度、時間割合、海氷厚、積雪深、海氷温度、海氷流速、海氷速度、海氷生成率
    • 30分毎(瞬間): 海面高度、海面水温・塩分・流速
    • 1時間毎: 験潮所海面高度
    • 全出力のリスト(PPTファイル)

  • 地形(grads形式): JPN, NP

把握している不具合、バイアス等について

  • 2008年2月19日頃-2008年3月7日頃: 水温過大。特に亜熱帯域。
  • 2011年-2012年: 現場観測データの未同化(SSH, SSTは同化)により、他年より解析精度が劣る。
  • 観測では存在しない海氷
    • 親潮域:20080222-20080823, 20090331-20090406, 20100309-20100317, 20120501-20120519, 20140409-20140411, 20150228-20150302, 20150320-20150322, 20160308-20160310,
    • 陸奥湾:201002-201003, 201402

  • オホーツク海南部の亜表層の高温バイアス
  • 黄海の夏季底層の高温バイアス
  • 日本海深層の流速

海面高度の海況変動の計算方法

JPNには海面気圧による押し下げ・吸い上げと潮汐が導入されているため、モデル出力海面高度 h には、海況要因による潮位偏差 h_ocean に加えて天文潮位 h_tide 、気圧応答による h_slpa も含まれる。 

h=h_tide + h_slpa + h_ocean

よって、海況変動の海面高度を推定する場合は、以下の式を用いる。 

h_ocean = h - h_tide - h_slpa

h_tide:モデル内潮汐場の8分潮調和解析

モデル出力の30分毎SSHスナップショット(hst_30min-jpn/hs_snp_ssh)を用いて調和解析し、8分潮の調和定数から h_tide を計算した。

  • h_tide(日平均): tideh.YYYYMMDD

h_slpa:海面気圧偏差によるinverted barometer

Inverted barometer 応答の仮定により、-1cm/hPaを用いる。 ローカルな気圧応答 h_slp (=-P(x,y,t)) と全球平均 h_slpglb (=-P_全球平均(t)) の差で与える。 

h_slpa = h_slp - h_slpglb = P_全球平均(t) - P(x,y,t)
  • h_slp(日平均): hs_slp.YYYYMMDD
  • h_slpglb(日平均): hs_slp_ave.YYYYMMDD
