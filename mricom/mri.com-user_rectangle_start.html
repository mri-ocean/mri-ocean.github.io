矩形海モデル・パッケージ「rectangle」

シンプルな設定の矩形海モデル。MRI.COMのテストに使われており、パッケージ中に最新のMRI.COM(開発版)が含まれる。

概要

  • 水平解像度1°、水深6000mの矩形海で理想的なダブルジャイア実験を行う。
  • グリッド数は62×52×10であり、PCでも動かせる程度に小さい。
  • デフォルトでは、ダブルジャイア用の東西風の下で10日間の積分を行い、初期順圧応答を見る。
  • 詳細な仕様
  • MRI.COM開発における単体テストや統合テストも本パッケージで行っている。

デモ

  • デフォルト設定の実験結果：海面高度の時間発展

バージョン

MRI.COMに合わせて、月1回更新します。

  • リリースノート

必要要件

  • Fortran コンパイラ
  • MPIライブラリ (設定を変えればMPI並列なしでも動く)
  • make
  • GrADS (他の描画ツールでも良い)
  • git

動作実績

  • スパコン
    • 気象研 FX-100, SR-16000
    • 東大FX10
    • JAMSTEC ICE-X
    • ES3 (SX-ACE)
    • 北大 SR-16000
  • サーバー
    • 気象研海洋 (Xeon + SUSE Linux + PGI Fortran)
  • PC
    • Ubuntu Linux + gfortran
    • Debian + gfortran (Google Compute Engine)
    • CentOS7 + gfortran (ZSTARオプションなし+MPI並列時に落ちる現象が確認されています。ZSTARの使用を推奨。)

提供物と取得方法

  • パッケージ(ソースコード環境一式)
    • 気象庁、気象研の方：本庁Redmineを参照
    • その他の方：GitHub MRI.COM-rect
      • 事前に利用申請とGitHub登録が必要です
  • 入力ファイル一式 GitHubでホスト中(今のところ)
    • > wget https://github.com/mokkei1978/rectangle_data/archive/master.zip; unzip master.zip

使い方

  • デフォルト設定で動かす
  • 動かない場合の対応
  • 設定を変更する
  • rectangleパッケージにおける設定ファイル

開発に参加するには

  • 気象研、気象庁の方：本庁Redmineに参加
  • モデル共同研究の方：DIAS Redmineに参加
  • その他の方：開発者(坂本)まで連絡してください
