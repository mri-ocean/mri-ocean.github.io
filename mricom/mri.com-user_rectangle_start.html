矩形海モデル・パッケージ「rectangle」

シンプルな設定の矩形海モデル。MRI.COMのテストに使われており、パッケージ中に最新のMRI.COM(開発版)が含まれる。

概要

目的

  • MRI.COM開発における単体テストや統合テストのためのモデル
  • 水平解像度1°、水深6000mの矩形海で理想的なダブルジャイア実験を行う
  • 様々なオプションの切り替えが可能

管理

  • 開発者：坂本圭 ksakamot@mri-jma.go.jp (気象研 全球4研)
  • 誰でも利用可。開発依頼等は坂本まで。
  • 開発状況: Redmine (気象研・気象庁のみ閲覧可)
  • 質問など: GitHub issues (要GitHub登録)

モデル仕様

  • グリッド数は62×52×10であり、PCでも動かせる程度に小さい
  • デフォルトでは、ダブルジャイア用の東西風の下で10日間の積分を行い、初期順圧応答を見る。
  • 詳細な仕様
  • デフォルト設定の実験結果：海面高度の時間発展

バージョン

MRI.COMに合わせて、月1回更新する。

  • リリースノート (GitHub) (要GitHub登録)
  • 過去のリリースノート

モデルの実行

必要要件

  • Fortran コンパイラ
  • MPIライブラリ (設定を変えればMPI並列なしでも動く)
  • make
  • GrADS (他の描画ツールでも良い)
  • git

提供物と取得方法

  • パッケージ(ソースコード環境一式)
    • 気象庁、気象研の方：本庁Redmineを参照
    • その他の方：GitHub MRI.COM-rect (事前に利用申請とGitHub登録が必要)
  • 入力ファイル一式 GitHubでホスト中(今のところ)
    • > wget https://github.com/mokkei1978/rectangle_data/archive/master.zip; unzip master.zip

ライセンス

MRI.COMにならいます。(第三者への提供や一般公開は許可していません。)

使い方

  • 実験の手順 - exp/README.md を参照
  • 動かない場合の対応
  • 設定を変更する
  • rectangleパッケージを利用したMRI.COMの開発
  • rectangleパッケージにおける設定ファイル
