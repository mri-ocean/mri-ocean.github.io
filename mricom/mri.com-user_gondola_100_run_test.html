テスト実験のチュートリアル

テスト設定 (16MPI並列、24step) で実験を行う手順です。

パッケージの展開

パッケージのtar ballを展開するか、gitでパッケージを取得する。 

> git clone -b (ブランチ名) [共有リポジトリURL]

ブランチ名は例えば master など。 以下、展開したディレクトリを$TOPと表記する。

実験の実行と結果の検証

気象研海洋サーバ (ogsv009) および気象研スパコンの場合

  1. $TOP/test に移動
  2. [実験実行] “./exec.sh (実験モード) (実験名)” を実行
  3. [結果検証] “./check.sh (実験モード) (実験名) (参照実験名)” を実行

実験モードについては、$TOP/test/READMEを参照のこと。

例えば、実験モード=parallel、実験名=next、参照実験名=masterの場合 

$ cd $TOP/test
$ ./exec.sh parallel next
$ ./check.sh parallel next master

とする。

./check.sh parallel next master では、$TOP/compare_next_parallel_to_master_parallel.txt が生成される。 全ての項目で [o.k.] が出力されていれば、2つの実験の結果の一致がわかる。
