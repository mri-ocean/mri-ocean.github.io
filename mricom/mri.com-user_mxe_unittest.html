MXEの単体テスト

MXEのPythonモジュールには単体テストを作成する。単体テストは tests/ の下に置き、make check で実行できるようにする。単体テストのコードは python unittest を使う。

実例

MXE anlpy/particle/ に単体テストの実例がある。

事前の準備

使うための準備

  • MXEの更新、mxe-data の更新
  • pythonライブラリのインストール: matplotlib, numpy, scipy, pandas, cartopy, cmocean

作成するファイル

下の順番でファイルを見てください。

  1. README-MXE.md
  2. Makefile
  3. sample/plot_particle.py - 絵を描くスクリプト(linkdir/mxe-data/temp/に出力)
  4. particle/particle.py - particle用ライブラリ・モジュール
  5. tests/test_particle.py - particle.pyの単体テスト・モジュール
