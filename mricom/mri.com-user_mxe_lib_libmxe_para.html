libmxe_para

実験の基本的なパラメータを設定するモジュール。 使い方はMXEライブラリを用いたプログラミングを参照してください。

構造体の型

  • type_libmxe_para: 実験パラメータの情報を保持するクラス。要素はソースコードを参照。

サブルーチン

libmxe_para__register

namelistファイルを読んで、クラスtype_libmxe_paraのインスタンスを生成するメソッド。

デフォルトでは./NAMELIST.MXEを参照しますが、オプショナル引数file_namelistでファイルのパスを指定することもできます。

下がサンプル・プログラムです。NAMELIST.MXEとNAMELIST.MXE-2という2つの実験設定についてインスタンスを作成します。 

program main
use libmxe_para,only: type_libmxe_para, libmxe_para__register
type(type_libmxe_para) :: para, para2
call libmxe_para__register(para)
call libmxe_para__register(para2,file_namelist="NAMELIST.MXE-2")
end program

定数

よく使われる定数を定義しています。例えば

  • pi=3.14…: パイ
  • clen=256: 文字変数の長さ

詳細はソースコードを参照してください。
