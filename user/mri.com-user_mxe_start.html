MRI.COM実行環境 MXE

MRI.COM実行環境 (MRI.COM eXecution Environment; MXE)とは、MRI.COM実験の前処理、実行、後処理、解析のプログラムを集約したパッケージです。 MRI.COM利用者の共通基盤となることを目指しています。

特徴

  • MRI.COMの前処理、後処理、解析ツールを集約
  • いくつかの設定込みパッケージのベースに使われている
  • 地形の読み込みなどのルーチンをライブラリとして提供
  • コミットルールの違いから、MRI.COMとは別パッケージとして管理
  • いくつかのツールは単体テストによって動作が確認されている

必要要件

  • Fortranコンパイラ (MPIは使わない)
  • make

動作実績

矩形海モデル、全球渦なしモデル GONDOLA_100、日本沿岸モデル、瀬戸内海モデルの開発に使われている。

取得方法

  • 気象庁、気象研の方：本庁Redmineを参照
  • モデル共同研究の方：JAMSTEC Redmineを参照
  • その他の方：MRI.COMのおまけとして提供されます

使い方

  • MXEのツールを使う
  • MXEをベースに実験パッケージを作る
  • 実験パッケージのMXEをアップデートする

内容

各ディレクトリの内容です。詳細は README-MXE.md を参照。

  • exp 実験の実行
  • prep 前処理
  • postp 後処理
  • anl 解析
  • setting 設定
  • lib MXE独自のFortranライブラリ
  • linkdir データディレクトリへのシンボリックリンク
  • grads/ gradsスクリプトを置きます。
  • doc/ 実験に関するドキュメントを置きます。

バーション

MRI.COMに合わせて月1回の更新。 更新内容は気象庁Redmine ロードマップを参照。
