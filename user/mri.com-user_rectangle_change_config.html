実験設定の変更

デフォルトではダブル・ジャイア用の風だけの非常にシンプルな設定ですが、設定を色々と変えることができます。 以下の3つの変更方法があります。

時間ステップ、積分期間、出力等の変更

デフォルトでは時間ステップ間隔30分で10日間の積分を行います。 時間ステップ、積分期間、出力を変更するには、exp/run/run.conf を編集し run_pre.sh を実行します。

実験オプションの変更 (自動)

ある特定の設定変更については、必要なファイル変更を自動で行うシェル・スクリプトがあります。 

> cd exp/run
> sh change_option.sh  !!! 実験設定名をリスト表示する
> sh change_option.sh [設定名]   !!! 設定名で設定ファイルを書き換える

元に戻すには、gitの機能を使ってください。

オンライン・ネスティング使用時の注意

オンライン・ネスティング使用時にはSCUPライブラリを使用します。 パッケージに同梱されたSCUPライブラリをコンパイルしておきます。 

> cd exp/scup/src
> make FC_XXX=マシン環境名  !!! Makefileを直接に書き換えても良い

実験オプションの変更 (手動)

手動で設定を変えるには、主に下の2つを編集します。

  • exp/config_files/configure.in
  • exp/run/namelist/ 以下のNAMELISTテンプレートファイル、またはrun_pre.shで生成されるNAMELIST.OGCM

詳細は、exp/MRICOM3/docs/README.Options, README.Namelist などを参照して下さい。
