瀬戸内海モデル MRI.COM-Seto

気象研で開発している瀬戸内海領域モデルのパッケージ。最新のMRI.COMを用いた実験設定や、前処理ツール、解析ツールをまとめています。

本庁へ納入した MOVE/MRI.COM-Seto からはZSTAR使用、沿岸地形の向上を施しています。

概要

  • 水平解像度1/33°×1/50°(約2km)の瀬戸内海領域モデル。
  • フォーシングにはGSMとMSM、側面境界にはMOVE-WNP-4DVARを使用。
  • 海面気圧による押し下げ吸い上げと、潮汐は与えていない。
  • 2009-2011の3年間のフリーランによってチューニングしている。
  • 詳細な仕様

デモ

  • 実験結果：海面水温の時間発展(2011)

バージョン

MRI.COMに合わせて、月1回更新します。

  • リリースノート

必要要件

  • Fortran コンパイラ
  • MPIライブラリ
  • make
  • GrADS

モデル動作実績

  • 気象研海洋サーバー(Xeon + SUSE Linux + PGI Fortran)
  • 気象研スパコン FX-100
  • 東大FX10

提供物

  • パッケージ　ソースコードと解析ツール等一式
  • 入力ファイル一式　(内容は同梱のREADME.mdを参照。入力ファイルは全て Big Endian。約35GB)
  • 実験結果 気象庁内のみ

取得方法

  • 気象庁、気象研の方：本庁Redmineを参照
  • モデル共同研究の方：DIAS Redmine MRI.COM Wikiを参照
  • その他の方：開発者(坂本)まで連絡してください

使い方

  • 標準実験のチュートリアル
  • 解析ツールを使う

開発に参加するには

  • 気象研、気象庁の方：本庁Redmine - 瀬戸内海2kmモデルに参加

ライセンス

気象庁内、及び共同研究の下でのみ使用可能です。
